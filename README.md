
# 🏠 Home Assignments – IoT Projects

This repository contains my IoT home assignment projects developed using Arduino and ESP32. Each project demonstrates the use of different sensors, actuators, and communication modules for real-world applications.

## 📂 Repository Structure

| Folder | Project | Description |
|---------|----------|-------------|
| `p1-climate-monitor` | Smart Climate Monitor | Monitors temperature and humidity using DHT22 and displays data on an OLED with buzzer alerts. |
| `p2-gas-fire-alert` | Gas & Fire Alert System | Detects gas leakage and fire, activates buzzer and warning LEDs. |
| `p3-parking-sensor` | Smart Parking Sensor | Measures distance using an ultrasonic sensor to assist parking. |
| `p5-bt-light-control` | Bluetooth Light Controller | Controls LEDs using Bluetooth commands from a mobile phone. |
| `p8-pressure-logger` | Pressure & Altitude Logger | Logs pressure and altitude readings using a BMP280 sensor. |
| `p10-home-hub` | Smart Home Hub | Integrates multiple sensors and devices into a simple home automation system. |

## 🚀 How to Use

1. Clone this repository.
2. Open the required project folder.
3. Open the `.ino` file using Arduino IDE.
4. Install the required libraries.
5. Upload the code to the Arduino/ESP32 board.
6. Test the project according to the project README.
